// PRIVACY

Your data never
leaves your machine

This is not a cloud service. Shield has no servers, no accounts, no telemetry, and no way to transmit your data — because we never built any of that infrastructure.

Everything runs locally on your CPU. The detection engine is a JavaScript file on your hard drive. You can read every line of it.

  • No account required, ever
  • No clipboard content stored beyond your session
  • No network requests made by the scanner
  • No analytics or usage tracking
  • Fully open source — audit every line
  • Works completely offline
